# HW
## HW1
* Arithmetic
    * Digit0 to Digit3 are initialized with the last 4 digits of your student id.
    * The program is to merge the four digits into MyID. 

## HW2
* Procedure
    * 用學號最後一碼初始化ChStrs變數
    * 寫一個procedure，將ChStrs中的字元轉換為0-1 bit(空白轉換為0，#號為1) 。
    * 使用Loop指令，呼叫前一項的procedure八次。
    * 印出轉換過後的結果至螢幕上
